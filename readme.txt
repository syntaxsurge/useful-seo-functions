== Description ==

Useful SEO Functions by SyntaxSurge optimizes and secures your WordPress site with features like:

- **SERPcraft.com Autoposting:** Automate SEO content posting directly from SERPcraft.com.
- **Affiliate Settings:** Manage Amazon Affiliate links, including domain switching and auto-insertion of product cards.
- **Security Settings:** Customize WordPress API endpoints and enhance site security.
- **SEO Settings:** Rank Math Meta Description integration, FAQ schema shortcodes, and auto-updating SEO keywords.

**Note:** The 'Disable Default WP API Prefix' setting is disabled by default to ensure compatibility with plugins like Elementor and Site Kit by Google.

== Frequently Asked Questions ==

= What is SERPcraft.com Autoposting and how does it work? =
SERPcraft.com Autoposting is a feature that automates the posting of content directly from SERPcraft.com to your WordPress site. Once configured, it can fetch and post content based on predefined settings, saving you time and effort in content management.

= How do I manage my Amazon Affiliate settings with this plugin? =
You can manage your Amazon Affiliate settings by navigating to the Affiliate Settings section of the plugin. Here, you can input your Amazon Affiliate ID, select your preferred Amazon domain, and configure settings like auto-inserting product cards and replacing existing affiliate tags.

= Is this plugin compatible with page builders like Elementor? =
Yes, this plugin is compatible with page builders like Elementor. However, if you're using such tools, ensure that the 'Disable Default WP API Prefix' setting is not enabled to maintain compatibility.

= Can I customize the SEO settings for my website using this plugin? =
Absolutely! The SEO Settings feature allows you to integrate with Rank Math Meta Description, use FAQ schema shortcodes, and automatically update SEO keywords. These tools are designed to optimize your website for better search engine visibility.

= What should I do if I encounter issues with the plugin? =
If you encounter any issues, please first check our documentation and existing FAQs for solutions. If the problem persists, you can reach out to our support team through the WordPress support forum or directly via our website.

= Are there any specific server requirements for this plugin? =
The plugin requires a server running PHP 7.4 or higher. It's also designed to be compatible with WordPress 5.9 and up, ensuring smooth performance on modern WordPress installations.

= How often is Useful SEO Functions by SyntaxSurge updated? =
We regularly update the plugin to introduce new features, fix bugs, and ensure compatibility with the latest WordPress versions. It's recommended to keep the plugin updated for optimal performance and security.
